# Financial Accounting Project (End-to-End Analytics)
## Project Goal
The goal of this project is to analyze corporate financial data (Revenue, Operating Costs, and Net Profit) using a modern data stack to identify savings opportunities, evaluate automation efficiency, and track regional performance.
## Tools & Technologies Used
- Database:PostgreSQL (pgAdmin) for secure data storage.
- Data Analytics:Python (Pandas, Seaborn, Matplotlib) in PyCharm Pro for statistical summary and correlation analysis.
- Visualization:Power BI for building an interactive executive dashboard.
## Core Insights from Python & Power BI
1. Expense Drivers:The Department-wise analysis shows specific high-operating-cost zones where cost-cutting can be implemented.
2. Profit Margins:Regional performance analysis highlights the top-performing sectors contributing to net profitability.
3. Data Trends:A strong correlation was found between Automation Efficiency and Net Profit metrics.
